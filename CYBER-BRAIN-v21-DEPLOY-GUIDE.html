API Reference — Malware Analysis Lab
REST endpoints with webhook support for sandbox orchestration
Submission APIs
POST
/api/v1/malware/submit
Submit file for analysis. Returns job_id. Supports multipart/form-data or JSON with base64. Hash-dedup included.
GET
/api/v1/malware/status/{job_id}
Poll analysis status. Returns: queued | running | completed | failed | deduped
GET
/api/v1/malware/report/{job_id}
Full normalized report. Includes IOCs, behavior, MITRE mapping, YARA/Sigma rules, AI verdict.
GET
/api/v1/malware/iocs/{job_id}
Extract IOCs only. IPs, domains, hashes, URLs, mutexes, registry keys.
POST
/api/v1/malware/hash-lookup
Check if hash already analyzed. Returns cached report if available. Saves sandbox credits.
Intelligence & Orchestration APIs
GET
/api/v1/intel/graph/{entity}
Get entity relationship graph. Supports pivot by IP, domain, hash, actor, campaign.
GET
/api/v1/memory/patterns
Get learned detection patterns from SOC Memory Engine. Sorted by confidence.
POST
/api/v1/agent/investigate
Trigger autonomous investigation. Returns job_id for async tracking.
POST
/api/v1/whatif/simulate
Run what-if simulation. Provide current attacker position + context.
POST
/api/v1/storyline/generate/{incident_id}
Generate attack storyline narrative from incident data.
Sandbox Provider Integration Example
// Sandbox Orchestrator — Cost-Aware Routing
const PROVIDERS = {
  joeSandbox:       { cost_per_scan: 0.50, capabilities: ['static','dynamic','network','memory'] },
  anyRun:           { cost_per_scan: 0.20, capabilities: ['interactive','network','process'] },
  hybridAnalysis:   { cost_per_scan: 0.00, capabilities: ['static','dynamic'], rate_limit: 200 },
  intezer:          { cost_per_scan: 0.30, capabilities: ['code-similarity','malware-family'] },
};

// Adaptive routing decision
function routeSample(sample, budget) {
  if (sample.hash_known)    return 'cache';          // FREE — cached result
  if (sample.risk_score < 30) return 'hybridAnalysis'; // LOW cost for low-risk
  if (budget > 1.0)          return 'joeSandbox';     // FULL analysis
  return 'anyRun';                                       // BALANCED default
}
